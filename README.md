使用方法:
在BubbleImageView中我定义了两种加载图片的方式，
一种是加载网络图片
```java  
   
load(String url, int res, int placeHolderPic)
   
```
一种是加载本地图片

```java  
   
setLocalImageBitmap(Bitmap bm, int res)
   
```

大家可以根据需要加载图片。其中的参数代码中都有详细介绍。
